# Buzzly

Buzzly is a modern social media platform where users can create, share, and discover short videos, photos, and stories. The app is designed to help people connect, express creativity, and build communities through engaging content.

## Features
- 📹 Short videos
- 📸 Photo sharing
- 💬 Comments and messaging
- ❤️ Likes and follows
- 🔍 AI-powered content discovery
- 🔔 Real-time notifications
- 🔒 Secure user authentication

## Tech Stack
- Flutter (Mobile App)
- Firebase (Authentication, Database, Storage)
- Node.js (Backend API)
- GitHub (Version Control)
